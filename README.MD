Repositorio enfocado en la implementación, configuración y administración de servidores Windows y Linux en entornos empresariales.

Tecnologías
- Windows Server
- Linux (Ubuntu / CentOS)
- Bash
- PowerShell

Contenido
- Configuración de Active Directory
- Gestión de usuarios y políticas (GPO)
- Servicios Linux (SSH, Nginx)
- Scripts de automatización
Arquitectura
Infraestructura híbrida con servidores Windows y Linux, aplicando buenas prácticas de seguridad y administración.
Objetivo
Demostrar habilidades en administración de sistemas, automatización y gestión de infraestructura.
